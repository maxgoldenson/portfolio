# Images — Linear Capacitive Encoder

Drop your photos here with these exact filenames:

| Filename | What to put there |
|---|---|
| `01-problem-context.jpg` | Existing sensor or mechanism showing the space constraint |
| `02-space-constraints.jpg` | Grinder mechanism showing where the encoder needs to fit |
| `03-pcb-layout.jpg` | PCB layout screenshot from EasyEDA, or schematic |
| `04-assembled-prototype.jpg` | Assembled encoder PCB in hand or on bench |
| `05-test-fixture.jpg` | Validation setup / test fixture |
| `06-installed.jpg` | Final encoder installed in the machine |
| `07-accuracy-plot.jpg` | Linearity or accuracy data plot (screenshot is fine) |

**Tips:**
- JPG or PNG both work — just keep the name exactly as above
- Landscape orientation (wider than tall) looks best in the 16:9 slots
- Aim for at least 1200px wide for crisp display
